Musical Fountain

Python 实现的音乐喷泉可视化程序（魔法钢琴）。通过音频分析驱动粒子特效，模拟随着音乐节奏跳动的喷泉效果。

🚀 快速开始

本项目提供两种主要的视觉模式，您可以根据需求选择运行不同的入口文件：

1. 蓝色单色模式
文件‌: blue.py
音域范围‌: C4 - B5
按键支持‌: 14 个白键
运行方式‌:
bash
python blue.py

特点‌: 纯净的蓝色视觉效果，适合专注聆听与简约视觉体验。

3. 彩色多键模式
文件‌: main.py
音域范围‌: C4 - B4
按键支持‌: 12 个黑白键
运行方式‌:
bash
python main.py

特点‌: 丰富的色彩变化，对应钢琴黑白键，视觉表现力更强。

🛠️ 校准指南

鉴于不同乐器的音质差异以及录音环境的环境噪音影响，为了获得最佳的识别效果，建议在首次使用或更换音源时进行校准。

运行校准程序:‌

bash
python calibrate.py


校准模式:‌

1.自动校准‌: 系统将自动扫描并校准 C4 到 B5 范围内的所有音符频率阈值。

2.手动校准‌: 允许用户对个别识别不准的音符进行微调，以提高特定音高的触发灵敏度。

⚙️ 自定义配置

在 blue.py（单色音乐喷泉程序）中，您可以自由调整以下参数以定制个性化的视觉效果：

颜色 (color)‌: 修改喷泉粒子的基础颜色。
粒子数量‌: 控制屏幕上同时存在的粒子密度。
粒子大小‌: 调整单个粒子的大小。
射速‌: 改变喷泉向上喷射的速度。
扩散偏移‌: 调整粒子在空中散开的范围和随机性。

提示‌: 打开对应的 .py 文件，在代码顶部的配置区域即可找到这些变量进行修改。
